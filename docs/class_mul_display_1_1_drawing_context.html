MulDisplay 1.0.0.0
MulDisplay is a .NET all in one (2D/3D) display API designed as a debugging/display tool. It can be used to easily display camera frames, 3D point clouds, 3D objects, 2D/3D overlays, etc..

ResponseMode ResponseMode [get, set]
 2D only. The minimum delay between 2 frames is determined by the DataRefreshDelay for both modes. Synchronous : The frame and its overlays will always be drawn, the BuildAndRefresh method will always wait for it to be drawn at least once before returning. Asynchronous : The BuildAndRefresh method will return instantly, removing any sort of delay coming from the API, however certain frames may not be drawn if BuildAndRefresh is called too often.
